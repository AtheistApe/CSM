\begin{enumerate}
 \item Let $\v{a} = \lv{5,2}$ and $\v{b} = \lv{-3,2}$.
       \begin{enumerate}
        \item Make a reasonably large and accurate sketch of these vectors, then draw the vector projections $\proj{a}{b}$ and $\proj{b}{a}$. Make estimates of these vector projections based on your drawings.
        \item Find the actual projections $\proj{a}{b}$ and $\proj{b}{a}$ and compare them with your estimates. How good were your estimates?
       \end{enumerate}
 \item If a constant force $\v{F}$ acts on an object so as to displace it a distance $d$ in the direction of the force, the work done by $\v{F}$ on the object is defined to be $\|\v{F}\| d$. If the constant force does not act along the direction of the displacement $\v{d}$, we define the work done by the force on the object as the product of the component of the force $\v{F}$ along the displacement $\v{d}$ times the distance $d = \|\v{d}\|$ the object is displaced.

       Find the work done by a force $\v{F} = \lv{2, -3, 6}$ acting on an object so as to displace it from the point $P(-1, 1, 0)$ to the point $Q(3, 5, 7)$.
\end{enumerate}
